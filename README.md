# Web Backend - Books API

## Endpoints
### PUT /api/books/{id}
Update books by ID; needs to be provided with all fields

### PATCH /api/books/{id}
Partially update books by ID; only provide fields that you want to change

### DELETE /api/books/{id}
Deletes book by ID; returns updated list of books
